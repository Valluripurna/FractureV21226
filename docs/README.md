## System Overview

FractureDetect AI is an advanced medical imaging solution that uses deep learning to detect bone fractures in X-ray images. The system features:

- **Multi-Model Detection**: Five different deep learning models for accuracy
- **Secure Authentication**: JWT-based user management with OTP support
- **Persistent Storage**: MongoDB integration for user data and reports
- **Professional Reporting**: PDF generation with medical recommendations
- **History Management**: Complete record of all analyses
- **Medical Assistant**: AI-powered chat for medical guidance
