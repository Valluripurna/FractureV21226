Place your unseen X-ray images in this directory for model accuracy comparison.

Supported formats: JPG, JPEG, PNG, BMP, TIF, TIFF

The compare_model_accuracy.py script in the scripts directory will evaluate all models
on these images and determine which one achieves the highest accuracy.

If you have ground truth labels, create a JSON file with the same name as your image
files (without extension) and place it in this directory.
